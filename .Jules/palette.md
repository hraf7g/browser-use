## 2025-05-15 - Robust Theme Switching and Global Focus States
**Learning:** In applications using `data-theme` attributes for manual theme switching, media queries like `@media (prefers-color-scheme: dark)` must be carefully managed to avoid conflicts. Mirroring dark theme variables in both a `[data-theme='dark']` selector and a `(prefers-color-scheme: dark)` media query (with a `:not([data-theme='light'])` guard) ensures that manual user preference correctly overrides system settings while still respecting system defaults. Additionally, global `:focus-visible` styles are essential for consistent keyboard accessibility across all interactive components.
**Action:** Always mirror theme variables in both data-attribute selectors and media queries when implementing manual theme toggles, and establish global focus-visible styles early in the `globals.css` to ensure universal accessibility.
